== Description ==


= What is woofav: =

woofav a plugin for woocommerce that allows users to create their own list of favorite products.
The following features will be added each product automatically if needed but can be anywhere on the screen using the shortcode you want to add.

1. Add or remove products from the list of favorites
2. Show all products favorites
3. Customizing Table View favorite products

= instruction of using WooFav : =

First of all, make sure that WooCommerce is installed and activated.

When you activate the plugin woofav, a page called woofav be made. Where products can be seen that the user likes.

Like in the Display button on the product before purchase button will be added automatically.


You can go to Settings Woofav button to select your tastes.

2. Activate the plugin through the 'Ramdom Ajax Post' screen in WordPress
